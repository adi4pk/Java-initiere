# Java - Initiere

Curs de Java de la zero. Toate clasele stau in `src/`, una per topic, cu metode
statice goale (`// TODO`). Cerintele sunt scrise direct in comentariul fiecarei
metode. Rulezi totul prin `Main.java`.

---

## Clasele

| Fisier                          | Metode | Subiect |
|---------------------------------|--------|---------|
| `Conditionale.java`             | `cond01` ... `cond12` | `if`, `else if`, `else`, operator ternar |
| `ConditionaleNivelMediu.java`   | `med01` ... `med20`   | Conditionale combinate, calculator, validari |
| `BuclaFor.java`                 | `for01` ... `for10`   | Bucla `for`, sume, multipli, puteri |
| `BuclaWhile.java`               | `while01` ... `while20` | Bucla `while`, cifre, factorial, prime, fibonacci |
| `Vectori.java`                  | `vec01` ... `vec10`   | Array-uri: parcurgere, max/min, sortare |
| `Liste.java`                    | `list01` ... `list10` | `ArrayList`: add/remove/contains/parcurgere |
| `Stringuri.java`                | `str01` ... `str10`   | `String`: lungime, contains, substring, palindrom |

---

## Cum lucrezi

1. Deschide folderul `adi-java-initiere/` in IntelliJ.
2. Alegi un topic (ex. `Conditionale.java`).
3. Citesti comentariul din metoda pe care vrei s-o lucrezi (ex. `cond01`).
4. Implementezi in interiorul metodei (declari variabilele si scrii codul).
5. In `Main.java` decomentezi linia `Conditionale.cond01();`.
6. Click Run pe `Main`.

> Doar UNA dintre liniile din `Main.java` ar trebui sa fie decomentata o data —
> ca sa vezi clar output-ul metodei la care lucrezi.

Valorile le pui DIRECT in cod, nu citesti de la tastatura.

---

## Ordine recomandata

```
Conditionale  ->  ConditionaleNivelMediu  ->  BuclaFor  ->  BuclaWhile
              ->  Vectori  ->  Liste  ->  Stringuri
```

Fiecare topic presupune ce ai invatat la cele anterioare.
